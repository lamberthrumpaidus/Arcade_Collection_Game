# 🕹️ Arcade Collection Game
### "Satu Portal, Puluhan Keseruan Klasik dalam Satu Genggaman"

> **Diciptakan dengan dedikasi oleh [Lamberth Rumpaidus](https://github.com/lamberthrumpaidus)**


**Arcade Collection Game** adalah sebuah platform hiburan digital yang menghadirkan **20 mini games** legendaris. Proyek ini bukan sekadar koleksi biasa, melainkan sebuah eksperimen **Modular JavaScript** yang dirancang untuk performa tinggi, kontrol presisi, dan antarmuka yang sangat bersih.

## 🌟 Fitur Unggulan

- **🤖 Smart Bot AI**: Lawan kecerdasan buatan pada *Tic Tac Toe* dan *Ular Tangga* yang punya logika menantang.
- **🎯 Pixel Perfect Hitbox**: Perbaikan total pada logika tabrakan (burung, mobil, dino). Nggak ada lagi istilah "tembus tembok" atau bug *immortal*.

## 🎮 Katalog Permainan (20-in-1)

| Kategori | Judul Permainan |
| :--- | :--- |
| **Strategi & Papan** | Tic Tac Toe (VS Bot), Ular Tangga, Sapu Ranjau |
| **Aksi & Refleks** | Snake Game, Tangkap Apel, Flappy Burung, Dino Lari, Balap Mobil, Tembak Pesawat |
| **Logika & Otak** | Cocok Kartu, Tebak Angka, Ingat Pola (Simon), Puzzle Geser, Tebak Warna |
| **Kasual & Keberuntungan** | Ping Pong, Pukul Tikus, Suit Jari, Tembak Target, Balap Klik, Slot Mesin |
